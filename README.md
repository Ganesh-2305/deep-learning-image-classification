# Deep Learning Image Classification Project

## Project Overview
This project implements a Deep Learning Image Classification model using TensorFlow and Convolutional Neural Networks (CNN). The model is trained on the CIFAR-10 dataset to classify images into multiple categories.

---

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

---

## Deep Learning Technique
Convolutional Neural Network (CNN)

---

## Dataset
CIFAR-10 Dataset

Classes:
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

---

## Features
- Image Classification

- Model Training

- Prediction System
- Model Saving
